点到直线距离法解题

已知：$A(0,0)$，$B(6,1)$，$C(3,6)$

1. 点到直线距离公式：$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$，其中 $Ax + By + C = 0$ 是直线方程

2. 求直线AB的方程：
• 斜率 $k_1 = \frac{1-0}{6-0} = \frac{1}{6}$
• 点斜式：$y - 0 = \frac{1}{6}(x - 0)$，即 $y = \frac{x}{6}$
• 一般式：$x - 6y = 0$

3. 求直线AC的方程：
• 斜率 $k_2 = \frac{6-0}{3-0} = 2$
• 点斜式：$y - 0 = 2(x - 0)$，即 $y = 2x$
• 一般式：$2x - y = 0$

4. 点C到直线AB的距离：
• 代入公式：$d_1 = \frac{|3 - 6 \times 6|}{\sqrt{1^2 + (-6)^2}} = \frac{|3 - 36|}{\sqrt{37}} = \frac{33}{\sqrt{37}}$

5. 点B到直线AC的距离：
• 代入公式：$d_2 = \frac{|2 \times 6 - 1|}{\sqrt{2^2 + (-1)^2}} = \frac{|12 - 1|}{\sqrt{5}} = \frac{11}{\sqrt{5}}$

6. 计算$\tan\angle BAC$：
• 在三角形ABC中，设$\angle BAC = \theta$
• 设$|AB| = a$，$|AC| = b$
• $d_1 = b \cdot \sin \theta$，$d_2 = a \cdot \sin \theta$
• 由余弦定理：$a \cdot b \cdot \cos \theta = \overrightarrow{AB} \cdot \overrightarrow{AC} = 24$
• $|AB| = \sqrt{6^2 + 1^2} = \sqrt{37}$，$|AC| = \sqrt{3^2 + 6^2} = \sqrt{45} = 3\sqrt{5}$
• 因此：$\tan \theta = \frac{\sin \theta}{\cos \theta} = \frac{11}{8}$

要点：

  • 点到直线距离公式是解题的关键
  • 通过余弦定理和几何关系建立夹角与距离的联系
  • 点到直线距离可以表示为向量的叉积与模的比值
  • 注意三角形中点、边、高之间的几何关系
  • 该方法使用了解析几何的思想，适合于坐标已知的情况
