方程组法解题

已知：$A(0,0)$，$B(6,1)$，$C(3,6)$

1. 求直线AB的方程：

直线AB通过点A(0,0)和B(6,1)，斜率为$k_{AB} = \frac{1-0}{6-0} = \frac{1}{6}$

直线AB的方程为：$y = \frac{1}{6}x$ 或 $x-6y=0$

2. 求过点C作AB的垂线CD的方程：

垂线斜率 $k_{CD} = -\frac{1}{k_{AB}} = -\frac{1}{\frac{1}{6}} = -6$

代入点C(3,6)，得垂线方程：

$y-6 = -6(x-3)$

$y-6 = -6x+18$

$y = -6x+24$

$6x+y = 24$

3. 求垂线CD与直线AB的交点D：

联立方程组：$\begin{cases} x-6y=0 \\ 6x+y=24 \end{cases}$

将第一个方程$x=6y$代入第二个方程：

$6(6y)+y=24$

$36y+y=24$

$37y=24$

$y=\frac{24}{37}$

代回求$x$：$x=6y=6 \cdot \frac{24}{37}=\frac{144}{37}$

所以交点D的坐标为$D(\frac{144}{37}, \frac{24}{37})$

4. 计算$\tan\angle BAC$：

在直角三角形CAD中，$\tan\angle CAD = \frac{CD}{AD}$

点C到点D的距离：$CD = \left|\frac{144}{37}-3\right| \cdot \sqrt{37} = \left|\frac{144-111}{37}\right| \cdot \sqrt{37} = \frac{33}{37} \cdot \sqrt{37} = \frac{33\sqrt{37}}{37}$

点A到点D的距离：$AD = \frac{24}{37} \cdot \sqrt{37} = \frac{24\sqrt{37}}{37}$

因此：$\tan\angle BAC = \tan\angle CAD = \frac{CD}{AD} = \frac{33\sqrt{37}/37}{24\sqrt{37}/37} = \frac{33}{24} = \frac{11}{8}$

要点：

  • 利用方程组求解几何问题的关键是正确建立直线方程
  • 通过联立方程组求出交点坐标
  • 利用点到直线距离公式计算垂直距离
  • 利用比值关系求出角度的正切值

垂线方程法解题

已知：$A(0,0)$，$B(6,1)$，$C(3,6)$

1. 求直线AB的方程：

斜率 $k_{AB} = \frac{1-0}{6-0} = \frac{1}{6}$

直线AB的方程为 $y = \frac{1}{6}x$ 或 $6y-x=0$

2. 过点C作AB的垂线方程：

垂线斜率 $k_{垂线} = -\frac{1}{k_{AB}} = -\frac{1}{\frac{1}{6}} = -6$

代入点C(3,6)得：$y-6 = -6(x-3)$

$y-6 = -6x+18$

$y = -6x+24$

$6x+y = 24$

3. 求垂线与AB的交点D：

联立方程 $\begin{cases} 6x+y = 24 \\ 6y-x = 0 \end{cases}$

从第二个方程得：$x = 6y$

代入第一个方程：$6(6y)+y = 24$

$36y+y = 24$

$37y = 24$

$y = \frac{24}{37}$

$x = 6y = 6 \cdot \frac{24}{37} = \frac{144}{37}$

所以交点D的坐标为 $D(\frac{144}{37}, \frac{24}{37})$

4. 计算距离CD和AD：

$CD = abs (\frac{144}{37}-3) \sqrt{37} = abs (\frac{144-111}{37}) \sqrt{37} = abs (\frac{33}{37}) \sqrt{37} = \frac{33\sqrt{37}}{37}$

$AD = abs (\frac{24}{37}) \sqrt{37} = \frac{24\sqrt{37}}{37}$

5. 计算比值CD/AD：

$\frac{CD}{AD} = \frac{\frac{33\sqrt{37}}{37}}{\frac{24\sqrt{37}}{37}} = \frac{33}{24} = \frac{11}{8}$

所以我们换一种方法计算

6. 另一种计算方法：

点C到直线AB的距离：$d = \frac{|6y-x|}{\sqrt{6^2+1^2}} = \frac{|6 \cdot 6 - 3|}{\sqrt{37}} = \frac{33}{\sqrt{37}}$

点A到直线CD的距离：$d_A = \frac{|6x+y-24|}{\sqrt{6^2+1^2}} = \frac{|6 \cdot 0 + 0 - 24|}{\sqrt{37}} = \frac{24}{\sqrt{37}}$

由于垂线CD和直线AB垂直相交于点D，所以：

$\frac{CD}{AD} = \frac{d}{d_A} = \frac{33/\sqrt{37}}{24/\sqrt{37}} = \frac{33}{24} = \frac{11}{8}$

要点：

  • 垂线方程法是一种有效的几何问题解法
  • 计算过程中要注意分数计算和化简
  • 利用点到直线距离公式可以简化计算
  • 垂线的斜率是原直线斜率的负倒数
