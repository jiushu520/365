已知：$B(0,0)$，$C(3,0)$，$A(3,1)$。点 $C'$ 是点 $C$ 关于直线 $AB$ 的对称点，求点 $C'$ 的坐标。

60

公式法解题

方法参考自 Bilibili 视频：https://www.bilibili.com/video/BV1Nj421f7jb?t=24.1

1. 确定直线 $AB$ 的方程：
• 直线经过 $B(0,0)$ 和 $A(3,1)$。
• 斜率 $m = \frac{1-0}{3-0} = \frac{1}{3}$。
• 方程为 $y = \frac{1}{3}x$，即 $x - 3y = 0$。
• 该方程形式为 $ax + by + c = 0$，其中 $a=1$, $b=-3$, $c=0$。

2. 确定待反射点 $C$ 的坐标：
• $C = (x_0, y_0) = (3, 0)$。

3. 应用点关于直线的对称点公式：
• 对于点 $(x_0, y_0)$ 和直线 $ax + by + c = 0$，其对称点 $(x', y')$ 满足： \[ \frac{x' - x_0}{a} = \frac{y' - y_0}{b} = -2 \frac{ax_0 + by_0 + c}{a^2 + b^2} \]
• 计算右侧的值：$k = -2 \frac{ax_0 + by_0 + c}{a^2 + b^2}$ \[ k = -2 \frac{(1)(3) + (-3)(0) + 0}{1^2 + (-3)^2} \] \[ k = -2 \frac{3 + 0 + 0}{1 + 9} \] \[ k = -2 \frac{3}{10} = -\frac{6}{10} = -\frac{3}{5} \]

4. 求解 $x'$ 和 $y'$：
• $\frac{x' - x_0}{a} = k \implies \frac{x' - 3}{1} = -\frac{3}{5}$ \[ x' = 3 - \frac{3}{5} = \frac{15 - 3}{5} = \frac{12}{5} \]
• $\frac{y' - y_0}{b} = k \implies \frac{y' - 0}{-3} = -\frac{3}{5}$ \[ y' = (-3) \times (-\frac{3}{5}) = \frac{9}{5} \]

5. 结论：
• 对称点 $C'$ 的坐标为 $(\frac{12}{5}, \frac{9}{5})$。

要点：

  • 直接应用点关于直线对称的通用公式。
  • 准确代入直线方程的系数 $a, b, c$ 和点的坐标 $x_0, y_0$。
  • 该公式结合了中点在直线上和连线垂直于直线的两个条件。
  • 适用于任何直线和点。
