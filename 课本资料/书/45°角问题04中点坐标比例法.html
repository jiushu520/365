已知：矩形$ABCD$中，$A(0,0)$，$B(6,0)$，$C(6,6)$，$D(0,6)$，$E(6,3)$，$G(-3,6)$，$F(a,6)$。点$F$在$E$与$G$之间，且$\angle FAE = \angle FAG$，求$a$的值。

中点坐标比例法解题

1. 根据题意，点$F(a,6)$在$E(6,3)$与$G(-3,6)$之间，且$\angle FAE = \angle FAG$。

2. 连接$GE$，求出$GE$的中点$M$的坐标：

\[ M = \left(\frac{x_G + x_E}{2}, \frac{y_G + y_E}{2}\right) = \left(\frac{-3 + 6}{2}, \frac{6 + 3}{2}\right) = \left(\frac{3}{2}, \frac{9}{2}\right) \]

3. 由于$\angle FAE = \angle FAG$，根据角平分线的性质，点$F$在$\angle EAG$的角平分线上。

4. 在这种几何配置下，当$\angle FAE = \angle FAG$时，点$F$、点$A$、点$M$（$GE$的中点）三点共线。

5. 因此，$F$、$A$、$M$三点的坐标满足共线条件。由于$A(0,0)$，$M\left(\frac{3}{2}, \frac{9}{2}\right)$，$F(a,6)$，

6. 计算$M$点坐标的横纵坐标比：

\[ \frac{x_M}{y_M} = \frac{\frac{3}{2}}{\frac{9}{2}} = \frac{3}{2} \times \frac{2}{9} = \frac{3}{9} = \frac{1}{3} \]

7. 由于$F$、$A$、$M$三点共线，且$A$为原点，所以$F$点的坐标比例应该与$M$点相同：

\[ \frac{a}{6} = \frac{1}{3} \]

8. 解方程：

\[ a = 6 \times \frac{1}{3} = 2 \]

9. 验证：当$a = 2$时，$F(2,6)$

  • $F$点坐标比：$\frac{2}{6} = \frac{1}{3}$
  • $M$点坐标比：$\frac{3/2}{9/2} = \frac{1}{3}$
  • 比例相等，验证正确

提示：

  • 中点坐标公式：$M = \left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}\right)$
  • 当$\angle FAE = \angle FAG$时，$F$在$\angle EAG$的角平分线上
  • 在此特殊配置下，$F$、$A$、$M$（$GE$中点）三点共线
  • 共线点的坐标比例相等：$\frac{x_F}{y_F} = \frac{x_M}{y_M}$
  • 最终答案：$a = 2$，横纵坐标比为$1:3$
