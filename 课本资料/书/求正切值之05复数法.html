复数法解题

已知：$A(0,0)$，$B(6,1)$，$C(3,6)$

1. 将点表示为复数：

  • $A = 0 + 0i$
  • $B = 6 + i$
  • $C = 3 + 6i$

2. 构造向量：

  • $\overrightarrow{AB} = B - A = 6 + i$
  • $\overrightarrow{AC} = C - A = 3 + 6i$

3. 计算复数相除：

$\frac{\overrightarrow{AC}}{\overrightarrow{AB}} = \frac{3 + 6i}{6 + i} = \frac{(3 + 6i)(6 - i)}{(6 + i)(6 - i)} = \frac{18 - 3i + 36i - 6i^2}{36 + 1}$

$= \frac{18 + 33i + 6}{37} = \frac{24 + 33i}{37}$

4. 计算复数的辐角θ，即 $\tan\angle BAC$：

$\tan\theta = \frac{\text{Im}(\frac{\overrightarrow{AC}}{\overrightarrow{AB}})}{\text{Re}(\frac{\overrightarrow{AC}}{\overrightarrow{AB}})} = \frac{33}{24} = \frac{11}{8}$

要点：

  • 复数法是处理平面几何问题的强大工具
  • 复数相除得到辐角差，即为向量夹角
  • 复数 $z = a + bi$ 的辐角可由 $\tan\theta = \frac{b}{a}$ 求得
  • 复数乘法对应向量旋转，复数除法对应求向量夹角
