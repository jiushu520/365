已知：矩形$ABCD$中，$A(0,0)$，$B(6,0)$，$C(6,6)$，$D(0,6)$，$E(6,3)$，$F(a,6)$。如图所示，$\angle FAE = 45°$，求$a$的值。

45°角问题解法：一线三直角弦图法

1. 构造全等三角形EGH，使其与三角形ABE全等：

  • 从点$E(6,3)$出发，向上作垂直线段$EG = 6$，得到$G(6,9)$
  • 从点$G(6,9)$出发，向左作水平线段$GH = 3$，得到$H(3,9)$
  • 连接$HE$，形成直角三角形$EGH$

2. 验证三角形$EGH$与三角形$ABE$全等：

  • 在$\triangle ABE$中：$AB = 6$，$BE = 3$，$\angle ABE = 90°$
  • 在$\triangle EGH$中：$EG = 6$，$GH = 3$，$\angle EGH = 90°$
  • 因此$\triangle ABE \cong \triangle EGH$（两直角边分别相等）

3. 分析角度关系：

  • 因为$\triangle ABE \cong \triangle EGH$，所以对应角相等
  • $\angle BAE = \angle GEH$
  • 延长$AE$到$H$，得到$AEH$为等腰直角三角形

4. 验证$\triangle AEH$为等腰直角三角形：

  • $A(0,0)$，$E(6,3)$，$H(3,9)$
  • $AE = \sqrt{6^2 + 3^2} = \sqrt{45} = 3\sqrt{5}$
  • $EH = \sqrt{(6-3)^2 + (3-9)^2} = \sqrt{9 + 36} = \sqrt{45} = 3\sqrt{5}$
  • $AH = \sqrt{3^2 + 9^2} = \sqrt{9 + 81} = \sqrt{90} = 3\sqrt{10}$
  • 因为$AE = EH$且$AH^2 = AE^2 + EH^2$，所以$\triangle AEH$为等腰直角三角形
  • $\angle AEH = 90°$，$\angle EAH = \angle EHA = 45°$

5. 利用45°角性质求解：

  • 因为$\angle FAE = 45°$，而$\angle HAE = 45°$
  • 所以$F$、$A$、$H$三点共线
  • 直线$AH$的斜率为：$k = \frac{9-0}{3-0} = 3$
  • 直线$AH$的方程为：$y = 3x$

6. 求交点坐标：

  • 点$F(a,6)$在直线$y = 3x$上
  • 代入得：$6 = 3a$
  • 解得：$a = 2$

7. 验证比例关系：

\[ \frac{AF}{AH} = \frac{AE}{AH} \cdot \frac{AF}{AE} \]

由于$A$、$F$、$H$共线，且$AF:FH = AE:EH$（相似三角形性质）

因此有比例：$3:9 = a:6$，解得$a = 2$

关键思路：

  • 构造全等三角形：$\triangle EGH \cong \triangle ABE$
  • 形成等腰直角三角形：$\triangle AEH$（$AE = EH$，$\angle AEH = 90°$）
  • 利用45°角性质：$F$、$A$、$H$三点共线
  • 建立比例关系：$3:9 = a:6$，得到$a = 2$
